# Value Crop

A comprehensive React-based frontend application for managing e-commerce inventory, products, and user dashboards. Built for speed and productivity.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Radix UI Components

## Getting Started

First, install the required dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

The application will be running at `http://localhost:5173`.
To build the application for production in a single standalone file, run `npm run build`.
